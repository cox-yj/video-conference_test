# 项目名称
这是一个视频会议项目，使用React、MobX和TypeScript开发。

## 开始
1. 克隆代码到本地：`git clone [repository-url]`
2. 进入项目目录：`cd [project-name]`
3. 安装依赖：`npm install`
4. 启动开发服务器：`npm start`

## 构建
运行以下命令构建项目：
npm run build



## 测试
运行以下命令进行单元测试：
npm run test


## 代码风格
使用ESLint进行代码风格检查，运行以下命令进行检查：
npm run lint

- React
- MobX
- TypeScript
